# TicTacToe

## Project Description
**TicTacToe** is a desktop game built with Python and the Turtle graphics library. It brings the classic Tic-Tac-Toe experience to life through a clean graphical interface, showcasing how event-driven programming and coordinate-based logic can create an interactive application from scratch.

## Installation and Running Instructions

### 1. Clone the repository
```bash
git clone https://github.com/NikitaHuslystyi/Tic_Tac_Toe.git
```

### 2. Navigate to the project folder
```bash
cd Tic_Tac_Toe
```

### 3. (Optional) Create a virtual environment

**Windows**
```bash
python -m venv venv
```

**macOS/Linux**
```bash
python3 -m venv venv
```

### 4. Activate the virtual environment

**Windows**
```bash
venv\Scripts\activate
```

**macOS/Linux**
```bash
source venv/bin/activate
```

### 5. No additional dependencies are required.

### 6. Run the project
```bash
python Tic_Tac_Toe.py
```

## Project Summary
**TicTacToe** explores the fundamentals of graphical Python development — mouse event handling, real-time rendering with Turtle, grid-based move tracking, and automatic win/draw detection — all packed into a lightweight and self-contained desktop app.
